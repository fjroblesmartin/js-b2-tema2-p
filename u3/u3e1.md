# Tema 2. Trabajo experto con clases

## Unidad 3. Clases abstractas

## Ejercicio 2.3.1 (Castellano)

Las clases abstractas son aquellas que están creadas con la intención de nos ser instanciadas.

Si bien JavaScript no cuenta con un mecanismo específico o palabra reservada para implementarlo, existen varias formas de lograrlo.

En este ejercicio vamos a crear una clase abstracta `Shape` que represente un figura geométrica cualquiera, y que además implementará un método abstracto `getArea`.

Para lograr la funcionalidad de clase abstracta y de método abstracto tendrás que:


### A. Clase `Shape`

Crear una clase que lleve por nombre `Shape`.


### B. Propiedad `nSides`

Dotarla de una propiedad `nSides`.


### C. Constructor

En el `constructor` lanzar un error en caso de que sea una instancia directa de la clase abstracta.

El error a lanzar es:

```js
throw new Error('ERROR_ABSTRACT. Abstract class Shape cannot be directly instantiated.');
```


### D. Método `getArea`

Crear un método `getArea` sobre el que se lanzará el siguiente error en caso de que sea llamado sin una implementación propia por
parte de la clase hija.

```js
throw new Error('ERROR_ABSTRACT_METHOD. Method "getArea()" should be implemented by child classes of "Shape".');
```


### E. Exportación de clase

Exportar la clase `Shape` para poder ser usada en otros módulos.

# Tema 2. Treball expert amb classes

## Unitat 3. Classes abstractes

## Exercici 2.3.1 (Català)

Les classes abstractes són aquelles que estan creades amb la intenció de ser instàncies.

Si bé JavaScript no compta amb un mecanisme específic o paraula reservada per implementar-lo, hi ha diverses maneres d'aconseguir-ho.

En aquest exercici crearem una classe abstracta `Shape` que representi una figura geomètrica qualsevol, i que a més implementarà un mètode abstracte `getArea`.

Per aconseguir la funcionalitat de classe abstracta i de mètode abstracte hauràs de:


### A. Classe `Shape`

Crear una classe que porti per nom `Shape`.


### B. Propietat `nSides`

Dotar-la d'una propietat `nSides`.


### C. Constructor

Al `constructor` llançar un error en cas que sigui una instància directa de la classe abstracta.

L'error a llançar és:

```js
throw new Error('ERROR_ABSTRACT. Abstract class Shape cannot be directly instantiated.');
```


### D. Mètode `getArea`

Crear un mètode `getArea` sobre el qual es llançarà el següent error en cas que sigui cridat sense una implementació pròpia per part de la classe filla.

```js
throw new Error('ERROR_ABSTRACT_METHOD. Method "getArea()" should be implemented by child classes of "Shape".');
```


### E. Exportació de classe

Exportar la classe `Shape` per poder ser utilitzada en altres mòduls.
